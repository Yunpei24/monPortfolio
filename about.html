My first Analysis and Reporting Project with Power BI

This Analysis and Reporting project, carried out using Power BI software, provides a solution to enable a university to effectively manage late arrivals and absences of lecturers and students. By focusing on proactive management and accurate monitoring, this application provides crucial information to improve course scheduling and ensure optimal student follow-up. Thanks to this Analysis and Reporting project with Power BI, the university benefits from a powerful tool for improving the management of lecturers' timetabling and improve student monitoring. By quickly identifying attendance problems, the university can take preventive and proactive measures to ensure an optimal learning learning environment for everyone involved.

Dashboard for viewing data retrieved from mongoDB Atlas using an API

This Data Visualisation Dashboard (developed with streamlit) that I developed offers an immersive and informative experience by dynamically presenting data retrieved from MongoDB Atlas, thanks to a custom API that I also designed with FastApi. This comprehensive and interactive solution enables users to visualise and explore the information stored in the MongoDB database in a simple and effective way. The MongoDB Atlas Data Visualisation Dashboard is a powerful tool for transforming raw data into actionable information. By providing interactive visualisations, insightful analysis and real-time connectivity to MongoDB Atlas, this application promotes more informed decision-making and helps users discover hidden trends and growth opportunities within their data.
